Column space & Null space

  • A is a m×nm \times nm×n matrix
  • There are two important subspaces of A:
    • Column space c(A)c(A)c(A)
      • which is in RmR^mRm
      • C(A)={Ax∣C(A) = \{Ax |C(A)={Ax∣ any xxx in Rn}R^n\}Rn}
    • Null space N(A)N(A)N(A)
      • which is in RnR^nRn
      • N(A)={x∣Ax=0}N(A) = \{x |Ax = 0\}N(A)={x∣Ax=0}
  • We said that Ax=bAx = bAx=b is solvable if and only if b can be expressed as combination of columns of A
    • 換句話說，b要落在C(A)C(A)C(A)(AAA的column space)中
    • C(A)C(A)C(A)就是spanspanspan(((vectors in A)A)A)
  • Nullspace of AAA (N(A)N(A)N(A))
    • The set of all solutions to Ax=0Ax = 0Ax=0
    • We know that N(A)N(A)N(A) is a subspace because:
      • Let’s suppose xxx, x′x'x′ are two vector in N(A)N(A)N(A), that is Ax=Ax′=0Ax = Ax' = 0Ax=Ax′=0
      • A(x+x′)=Ax+Ax′=0+0=0A(x+x') = Ax+Ax'= 0 + 0 = 0A(x+x′)=Ax+Ax′=0+0=0
      • A(cx)=c(Ax)=c⋅0=0A(cx) = c(Ax) = c · 0 = 0A(cx)=c(Ax)=c⋅0=0
      • Ax=0Ax = 0Ax=0 has infinite solutions iff N(A)≠{0}N(A) \ne \{0\}N(A)={0}
        • in other words, Ax=0Ax = 0Ax=0 has exactly one sol. if N(A)={0}N(A) = \{0\}N(A)={0}
      • Are all Ax=bAx = bAx=b(which b is not zero)'s solution a subspace?
        • NO
        • If the solution set of Ax=bAx = bAx=b has not path through the origin, that means we can combine two solutions of Ax=bAx = bAx=b, then generate a new vector which is not in the solution set of Ax=bAx=bAx=b

Reduced row echelon form

  • Is there a systematic way that can help us to find N(A)N(A)N(A)?
    • Yes.
    • Suppose:

A=[12342467361010]A=\begin{bmatrix}1&2&3&4\\2&4&6&7\\3&6&10&10\\\end{bmatrix} A=⎣⎢⎡​123​246​3610​4710​⎦⎥⎤​

  • Reduce A by Gauss Elimination，then get the UUU(upper triangular form) matirx

U=[123400120000]U=\begin{bmatrix}1&2&3&4\\0&0&1&2\\0&0&0&0\\\end{bmatrix} U=⎣⎢⎡​100​200​310​420​⎦⎥⎤​

  • We notice that U11U_{11}U11​ and U23U_{23}U23​ are 1，which are pivot variables
  • Let’s analyse UUU by formula’s perspective:

x1+2x2+3x3+4x4=0x3−2x4=0x_1+2x_2+3x_3+4x_4=0\\ x_3-2x_4=0 x1​+2x2​+3x3​+4x4​=0x3​−2x4​=0

  • Let’s say x2x_2x2​ and x4x_4x4​ are free variables
    • x2=αx_2 = \alphax2​=α, x4=βx_4 = \betax4​=β
    • So the pivot variable are:

  • Finally, we can get the Complete solution

[x1x2x3x4]=[−2α−10βα2ββ]\begin{bmatrix}x_1\\x_2\\x_3\\x_4\end{bmatrix} = \begin{bmatrix}-2\alpha-10\beta\\\alpha\\2\beta\\\beta\end{bmatrix} ⎣⎢⎢⎢⎡​x1​x2​x3​x4​​⎦⎥⎥⎥⎤​=⎣⎢⎢⎢⎡​−2α−10βα2ββ​⎦⎥⎥⎥⎤​

  • Let’s say that Complete solution is linear combination of Special solution, which is:

α[−2100]+β[−10021]=[−2−10100201][αβ]=Nc\alpha\begin{bmatrix}-2\\1\\0\\0\end{bmatrix} + \beta\begin{bmatrix}-10\\0\\2\\1\end{bmatrix}\\ = \begin{bmatrix}-2&-10\\1&0\\0&2\\0&1\end{bmatrix}\begin{bmatrix}\alpha\\\beta\end{bmatrix} = Nc α⎣⎢⎢⎢⎡​−2100​⎦⎥⎥⎥⎤​+β⎣⎢⎢⎢⎡​−10021​⎦⎥⎥⎥⎤​=⎣⎢⎢⎢⎡​−2100​−10021​⎦⎥⎥⎥⎤​[αβ​]=Nc

Independence, basis and dimension

  • v1, v2, …, vn are independent if
    • only the trivial combination gives zero vector.
      • c1v1+c2v2+...+cnvn=0c_1v_1+c_2v_2+...+c_nv_n = 0c1​v1​+c2​v2​+...+cn​vn​=0
      • c1=c2=...=cn=0c_1=c_2=...=c_n=0c1​=c2​=...=cn​=0
    • otherwise, they are dependent.
  • How to find the independent vectors in a span?
    • Reduce the matrix AAA to RRR，the column vector which has pivot is independent vector.
  • span(A)≠span(R)span(A) \ne span(R)span(A)=span(R)
  • We can also say that, a matrix is linear independet if
    • full column rank

    • v1,...,vnv_1,...,v_nv1​,...,vn​ are independent
